# Machine Learning Enabled Smart System for Visually Impaired People

A real-time assistive system designed to help visually impaired individuals understand their surroundings using computer vision, voice interaction, and location-based services.

The system detects objects using YOLOv8, provides spoken feedback, allows voice-controlled navigation, and sends emergency alerts with live location when needed.

---

## 🚀 Key Features

- Real-time object detection using YOLOv8  
- Voice feedback for detected surroundings  
- Emergency voice command to send SMS with location  
- Navigation support using live geolocation  
- Works with webcam or IP camera  
- Fast and lightweight processing  

---

## 🧠 Technologies Used

- Python  
- YOLOv8 (Ultralytics)  
- OpenCV  
- Speech Recognition  
- Text-to-Speech (pyttsx3)  
- Twilio API  
- Geolocation APIs
